# 验证码功能说明

## 实现时间
2026-05-06

## 功能概述
为注册和登录页面添加图形验证码功能，防止批量注册、暴力破解和自动化攻击。

## 实现方案

### 技术选型
- **验证码库**: `svg-captcha`
- **验证码类型**: SVG 图形验证码
- **存储方式**: 内存存储（Map）
- **验证方式**: 一次性验证，用后即焚

### 验证码特性
- ✅ 4位字符长度
- ✅ 排除易混淆字符（0oO, 1lI等）
- ✅ 彩色字符
- ✅ 干扰线条
- ✅ 不区分大小写
- ✅ 5分钟有效期
- ✅ 一次性使用
- ✅ 点击刷新

### 2. 验证码生成API

**接口**: `GET /captcha`

**响应**:
```json
{
  "captchaId": "abc123def456",
  "captchaSvg": "<svg>...</svg>"
}
```

**实现**:
```javascript
const svgCaptcha = require('svg-captcha')

app.get('/captcha', (req, res) => {
  const captcha = svgCaptcha.create({
    size: 4,                    // 验证码长度
    ignoreChars: '0o1ilI',      // 排除易混淆字符
    noise: 2,                   // 干扰线条数
    color: true,                // 彩色字符
    background: '#f0f0f0',      // 背景色
    width: 120,
    height: 40,
  })

  const captchaId = Math.random().toString(36).substring(2) + Date.now().toString(36)

  captchaStore.set(captchaId, {
    text: captcha.text.toLowerCase(),
    timestamp: Date.now(),
  })

  res.json({
    captchaId: captchaId,
    captchaSvg: captcha.data,
  })
})
```

### 3. 验证码验证函数

```javascript
function verifyCaptcha(captchaId, captchaText) {
  if (!captchaId || !captchaText) {
    return false
  }

  const stored = captchaStore.get(captchaId)
  if (!stored) {
    return false
  }

  // 验证码只能使用一次
  captchaStore.delete(captchaId)

  // 检查是否过期（5分钟）
  if (Date.now() - stored.timestamp > 5 * 60 * 1000) {
    return false
  }

  // 不区分大小写比较
  return stored.text === captchaText.toLowerCase()
}
```

### 4. 登录接口集成

```javascript
app.post('/auth/login', (req, res) => {
  const { username, password, captchaId, captchaText } = req.body

  // 验证验证码
  if (!verifyCaptcha(captchaId, captchaText)) {
    return res.status(400).json({ error: '验证码错误或已过期' })
  }

  // 继续登录逻辑...
})
```

### 5. 注册接口集成

```javascript
app.post('/auth/register', (req, res) => {
  const { username, password, email, captchaId, captchaText } = req.body

  // 验证验证码
  if (!verifyCaptcha(captchaId, captchaText)) {
    return res.status(400).json({ error: '验证码错误或已过期' })
  }

  // 继续注册逻辑...

```

### 6. 验证码清理机制

```javascript
// 清理过期验证码
setInterval(() => {
  const now = Date.now()
  for (const [key, value] of captchaStore.entries()) {
    if (now - value.timestamp > 5 * 60 * 1000) {
      captchaStore.delete(key)
    }
  }
}, 60 * 1000) // 每分钟清理一次
```

### 3. JavaScript 逻辑

**加载验证码**:
```javascript
let loginCaptchaId = null
let registerCaptchaId = null

async function loadCaptcha(type) {
  try {
    const response = await fetch(`${SRC_URL}/captcha`)
    const data = await response.json()

    if (type === 'login') {
      loginCaptchaId = data.captchaId
      document.getElementById('login-captcha-img').innerHTML = data.captchaSvg
    } else {
      registerCaptchaId = data.captchaId
      document.getElementById('register-captcha-img').innerHTML = data.captchaSvg
    }
  } catch (error) {
    console.error('加载验证码失败:', error)
    toastr.error('加载验证码失败')
  }
}

// 页面加载时加载验证码
loadCaptcha('login')
loadCaptcha('register')

// 点击刷新验证码
document.getElementById('login-captcha-img').addEventListener('click', () => {
  loadCaptcha('login')
})

document.getElementById('register-captcha-img').addEventListener('click', () => {
  loadCaptcha('register')
})
```

**登录提交**:
```javascript
document.getElementById('login-form').addEventListener('submit', async (e) => {
  e.preventDefault()

  const username = document.getElementById('login-username').value.trim()
  const password = document.getElementById('login-password').value.trim()
  const captchaText = document.getElementById('login-captcha').value.trim()

  if (!captchaText) {
    toastr.error('请输入验证码')
    return
  }

  const response = await fetch(`${SRC_URL}/auth/login`, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({ 
      username, 
      password, 
      captchaId: loginCaptchaId, 
      captchaText 
    }),
  })

  const data = await response.json()

  if (!response.ok) {
    // 验证失败，刷新验证码
    document.getElementById('login-captcha').value = ''
    loadCaptcha('login')
  }
})
```

## 验证码示例

### 生成的验证码样式
```
┌──────────────┐
│ a B 3 d      │  ← 4位字符
│  /  \  /     │  ← 干扰线条
│ 彩色字符     │  ← 不同颜色
└──────────────┘
```

### 特点
- 字符：a-z, A-Z, 2-9（排除0oO1lI）
- 颜色：随机彩色
- 干扰：2条干扰线
- 尺寸：120x40px
- 背景：浅灰色 #f0f0f0

## 存储机制

### 内存存储
```javascript
const captchaStore = new Map()

// 存储结构
{
  "captchaId": {
    "text": "ab3d",           // 验证码文本（小写）
    "timestamp": 1746547200000  // 创建时间戳
  }
}
```

### 优点
- ✅ 快速读写
- ✅ 自动过期
- ✅ 无需数据库

### 缺点
- ⚠️ 服务器重启丢失
- ⚠️ 多服务器不共享
- ⚠️ 内存占用

### 生产环境建议
使用 Redis 存储：
```javascript
// 使用 Redis
const redis = require('redis')
const client = redis.createClient()

// 存储验证码
await client.setEx(`captcha:${captchaId}`, 300, captcha.text)

// 验证验证码
const stored = await client.get(`captcha:${captchaId}`)
await client.del(`captcha:${captchaId}`)
```

## 错误处理

### 常见错误

#### 1. 验证码错误或已过期
```json
{
  "error": "验证码错误或已过期"
}
```

**原因**:
- 验证码输入错误
- 验证码已使用
- 验证码已过期（>5分钟）
- captchaId 不存在

**处理**:
- 前端自动刷新验证码
- 清空输入框
- 提示用户重新输入

#### 2. 请输入验证码
```json
{
  "error": "请输入验证码"
}
```

**原因**:
- 用户未输入验证码

**处理**:
- 前端验证，提示用户

#### 3. 加载验证码失败

**原因**:
- 网络错误
- 服务器错误

**处理**:
- 显示错误提示
- 提供重试按钮
