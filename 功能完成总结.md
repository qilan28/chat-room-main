## 🎯 核心功能说明

### 登录流程
```
1. 访问聊天室 → 检测未登录 → 跳转到 /login.html
2. 用户注册/登录 → 获取token → 存储到localStorage
3. 跳转回聊天室 → 验证token → 显示用户信息
```

### 通知流程
```
1. 用户被@或被回复 → 后端创建通知 → 保存到notifications.json
2. 前端定时获取通知 → 更新徽章数量
3. 用户点击通知按钮 → 显示通知列表
4. 点击通知 → 标记已读 → 更新徽章
```

### 私聊流程
```
1. 用户选择聊天对象 → 发送私聊消息
2. 后端保存消息 → 创建通知 → WebSocket推送
3. 接收方收到实时消息 → 更新私聊徽章
4. 点击私聊按钮 → 查看会话列表
```

## 🚀 使用方法

### 首次使用
1. 启动服务器：`npm start`
2. 访问：`http://localhost:3000`
3. 自动跳转到登录页面
4. 注册新账号或使用测试账号
5. 登录后进入聊天室

### 功能测试
1. **注册登录**
   - 访问 `/login.html`
   - 注册新用户（用户名3-20字符，密码至少6位）
   - 登录后自动跳转

2. **查看通知**
   - 点击右上角铃铛图标
   - 查看未读通知（红色徽章）
   - 点击通知标记为已读

3. **私聊**
   - 点击右上角对话图标
   - 查看私聊会话列表
   - （完整私聊功能需进一步实现）

4. **登出**
   - 点击侧边栏"登出"按钮
   - 确认后跳转到登录页

## 🔧 技术要点

### 认证机制
- 使用Bearer Token认证
- Token存储在localStorage
- 每个API请求携带Authorization头
- Token有效期7天

### 数据结构
```javascript
// 用户账号
{
  "id": "user1234567890",
  "username": "张三",
  "password": "123456",
  "email": "zhangsan@example.com",
  "avatar": "https://...",
  "createdAt": "2026-05-05T00:00:00.000Z"
}

// 通知
{
  "id": "notif123",
  "type": "mention|reply|private_message",
  "data": {
    "fromUsername": "李四",
    "messageId": "msg123",
    "text": "消息内容"
  },
  "timestamp": "2026-05-05T00:00:00.000Z",
  "read": false
}

// 私聊消息
{
  "id": "pm123",
  "fromUserId": "user123",
  "fromUsername": "张三",
  "toUserId": "user456",
  "text": "你好",
  "attachments": [],
  "timestamp": "2026-05-05T00:00:00.000Z",
  "read": false
}
```
