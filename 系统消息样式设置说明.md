# 系统消息样式自定义功能

## 功能说明

管理员现在可以通过后台管理面板自定义系统消息（如公告中的高亮文本）的样式，包括背景颜色和文字颜色。

## 使用方法

### 1. 访问设置界面

1. 登录管理员账户：`http://localhost:3000/admin-login.html`
2. 进入管理面板：`http://localhost:3000/message_manage`
3. 点击 **"聊天室设置"** 按钮

### 2. 自定义样式

在聊天室设置对话框中，找到 **"系统消息样式设置"** 部分：

- **背景颜色**：点击颜色选择器，选择您喜欢的背景颜色
- **文字颜色**：点击颜色选择器，选择您喜欢的文字颜色

### 3. 实时预览

在设置过程中，**"样式预览"** 区域会实时显示您选择的颜色效果，方便您调整到满意的样式。

### 4. 保存并应用

点击 **"保存设置"** 按钮，新的样式将立即应用到：
- 所有新发送的公告消息
- 聊天室中所有使用 `.sys_msg` 类的系统消息

## 技术实现

### 配置文件结构

样式配置保存在 `data/config.json` 文件中：

```json
{
  "roomName": "公共聊天室",
  "welcomeMessage": "欢迎来到聊天室！",
  "maxMessageLength": 5000,
  "systemMessageStyle": {
    "backgroundColor": "rgb(27, 128, 223)",
    "textColor": "pink",
    "borderRadius": "8px",
    "padding": "2px 6px"
  }
}
```

### 前端应用

前端会自动加载配置并动态应用样式：

```javascript
// 从服务器获取配置
const config = await fetch('/config')

// 动态创建CSS样式
const style = document.createElement('style')
style.textContent = `
  .sys_msg {
    color: ${config.systemMessageStyle.textColor} !important;
    background-color: ${config.systemMessageStyle.backgroundColor} !important;
    border-radius: ${config.systemMessageStyle.borderRadius} !important;
    padding: ${config.systemMessageStyle.padding} !important;
  }
`
document.head.appendChild(style)
```

## 使用示例

### 在公告中使用

在公告编辑器中，使用 `<h4 class="sys_msg">` 标签包裹需要高亮的文本：

```html
欢迎加入聊天室！你可以点击右上角的设置按钮自定义你的用户名和头像。
<br>
<h4 class="sys_msg">信息支持html标签哦~</h4>
```

### 效果展示

设置前（默认样式）：
- 背景色：蓝色 (rgb(27, 128, 223))
- 文字色：粉色 (pink)

设置后（自定义样式）：
- 背景色：您选择的颜色
- 文字色：您选择的颜色

## 颜色选择建议

### 推荐配色方案

1. **经典蓝粉**（默认）
   - 背景：#1B80DF (蓝色)
   - 文字：#FFC0CB (粉色)

2. **清新绿白**
   - 背景：#4CAF50 (绿色)
   - 文字：#FFFFFF (白色)

3. **活力橙黑**
   - 背景：#FF9800 (橙色)
   - 文字：#000000 (黑色)

4. **优雅紫白**
   - 背景：#9C27B0 (紫色)
   - 文字：#FFFFFF (白色)

5. **专业深蓝白**
   - 背景：#2196F3 (深蓝)
   - 文字：#FFFFFF (白色)

### 对比度建议

为确保文字清晰可读，建议：
- 深色背景搭配浅色文字
- 浅色背景搭配深色文字
- 避免使用对比度过低的颜色组合

## 注意事项

1. **颜色格式**：系统会自动将您选择的颜色转换为 RGB 格式存储
2. **即时生效**：保存后需要刷新页面才能看到新样式
3. **全局应用**：样式会应用到所有使用 `.sys_msg` 类的元素
4. **备份建议**：修改前建议备份 `data/config.json` 文件

## 常见问题

### Q: 修改后没有生效？
A: 请刷新浏览器页面（Ctrl+F5 或 Cmd+Shift+R）清除缓存。

### Q: 如何恢复默认样式？
A: 在颜色选择器中设置：
- 背景颜色：#1B80DF
- 文字颜色：#FFC0CB

### Q: 可以设置渐变色吗？
A: 当前版本仅支持纯色，不支持渐变色。

### Q: 样式会影响普通消息吗？
A: 不会，只影响使用 `.sys_msg` 类的系统消息。
